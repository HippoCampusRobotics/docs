BlueROV
#######

We have 2 BlueROV2s: 

- The old BlueROV. We use it mainly for the SEMS/FAV class. It's vehicle name is :code:`bluerov-01`.
- The newer BlueROV. This one is used for the UVMS setup and has the Alpha arm mounted to it. It's vehicle name is :code:`bluerov-02`.
